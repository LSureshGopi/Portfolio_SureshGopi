Main Projects

SKEW – Data Collection & Analytics Platform

End-to-end industrial data collection, visualization, and analytics system built with React and FastAPI.

SKEW is a comprehensive data collection and analytics platform developed for real-time industrial environments. It captures live process data from PLCs and IoT devices, processes it through a FastAPI backend, and stores it securely in PostgreSQL. The interactive React.js frontend provides dynamic dashboards to visualize trends, monitor production metrics, and analyze system performance. Users can perform detailed data analysis, generate reports, and export insights in various formats for operational decision-making. The system is fully containerized with Docker and version-controlled using Git, ensuring scalable, maintainable, and consistent deployments across environments.

SKEW ASRS – Automated Storage & Retrieval System

Smart warehouse automation system for real-time material storage, retrieval, and traceability.

SKEW ASRS is an intelligent warehouse management system designed to automate storage and retrieval operations using real-time data synchronization. Built with Flask for the backend and PostgreSQL for data handling, it enables seamless communication with PLCs and industrial controllers to execute automated material movements across storage cells. The system provides complete material traceability — tracking every product’s storage location, movement history, and retrieval process with high accuracy. Real-time dashboards allow operators to monitor inventory status, manage recipes, and optimize warehouse utilization. Versioned with Git and deployed using Docker, SKEW ASRS ensures scalable, reliable, and easily maintainable deployments across production environments.

SKEW Batch Reporting System

Automated batch reporting and recipe management system for process industries like feed mills and food manufacturing.

SKEW Batch Reporting is an end-to-end process reporting platform designed for production environments such as feed mills, food manufacturing, and chemical batching plants. Built using Flask and PyQt, it integrates seamlessly with PLC-based control systems to collect real-time batch data, monitor ingredient flow, and generate detailed production reports. The system enables recipe download, material traceability, and usage optimization to ensure consistency and reduce wastage. All data is managed in MSSQL for secure, high-volume handling, while deployment is powered by Git and Docker for efficient versioning and scalability. The software empowers operators and managers to track production history, analyze material usage trends, and optimize resource consumption for better process efficiency.

SKEW Energy Monitoring System

Real-time industrial energy analytics and optimization platform with FastAPI, React, and PostgreSQL.

SKEW Energy Monitoring is a high-performance system for tracking, analyzing, and optimizing industrial energy usage. Developed using FastAPI (backend) and React.js (frontend), it connects to PLCs and smart meters to collect live electrical parameters such as current, voltage, and power factor. Data is stored and analyzed in PostgreSQL, enabling detailed insights into energy consumption patterns and equipment performance. The platform provides intuitive dashboards for visualization, trend analysis, and load optimization, helping industries minimize energy costs and improve operational efficiency. The application is fully containerized with Docker, version-controlled via Git, and built for scalability and real-time responsiveness. Advanced analytics modules also support predictive maintenance and AI-driven load balancing for sustainable production.

Code-X – Automation Engineer Support Tool

Tool for rapid PLC program generation and automation workflow support.

Code-X is a productivity tool designed to assist automation engineers in quickly creating PLC programs, including function blocks (FB), timers (TL), and other logic components. Built with Flask for the backend and PostgreSQL for persistent configuration management, it streamlines repetitive coding tasks and enforces standardized programming practices. The tool integrates version control with Git, containerized deployments using Docker, and is deployed on AWS LightSail for scalable, cloud-based operation. Engineers can input structured project specifications and automatically generate PLC code, minimizing manual errors, accelerating project delivery, and improving overall operational efficiency.

MAIL - Email Marketing & Automation Tool

Automated email system for marketing campaigns, reports, and workflow optimization.

This Email Marketing & Automation Tool is designed to streamline communication and marketing processes by automatically sending emails, reports, and notifications based on schedules and triggers. Built with Python, it leverages SMTP and scheduling libraries to automate repetitive email tasks, while AWS ensures scalable delivery and cloud integration. The system is version-controlled with Git and deployed using Docker for reliable, containerized operation across environments. Key benefits include optimized campaign timing, reduced manual effort, and improved engagement analytics, allowing businesses to focus on strategy rather than execution.
