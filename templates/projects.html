SKEW is a comprehensive industrial analytics platform capturing live process data from PLCs and IoT devices.   • Real-time PLC & IoT data ingestion with live dashboard updates
  • Data analysis, report generation, and multi-format export
  • Fully containerized with Docker, versioned via Git
